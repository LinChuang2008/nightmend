REMEDIATE

Runbooks as first-class code

Author, review, and version runbooks in Git. VigilOps executes the approved runbook for the matched incident class — scoped to explicit guard-rails, with full audit trail and one-click rollback.

# runbooks/payments-api-pool.yml
match: incident.class == "pool_exhausted"
guard: env == "prod" and business_hours
action: restart.rolling(service, max=1)
